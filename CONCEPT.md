# Langley - Multi-Agent Crypto Intelligence Network

## Project Overview

**Langley** is a multi-agent AI system that functions as a **personal crypto intelligence agency**. 

It continuously monitors on-chain data, social sentiment, narratives, and market psychology to discover hidden alpha, detect risks, and deliver actionable insights. Unlike basic scanners or trading bots, Langley uses specialized agents that collaborate, debate, and cross-verify information like a real intelligence team.

**Tagline**: *Your Private Crypto Intelligence Agency*

## 🤖 Multi-Agent Architecture

The system is powered by a collaborative multi-agent framework (CrewAI / LangGraph):

### Core Agents:

1. **On-Chain Forensics Agent**  
   - Analyzes transactions, wallets, liquidity, and smart contracts
2. **Narrative Scout Agent**  
   - Tracks emerging trends on X, Telegram, and crypto communities
3. **Sentiment & Psychology Agent**  
   - Measures crowd emotion, whale activity, and FOMO/FUD cycles
4. **Risk Guardian Agent**  
   - Detects rugs, honeypots, malicious contracts, and portfolio risks
5. **Opportunity Simulator Agent**  
   - Runs market simulations and probability scenarios
6. **Synthesis Agent** (Orchestrator)  
   - Combines all findings into clear, ranked insights with confidence scores
7. **Storyteller Agent**  
   - Converts technical data into compelling, shareable narratives

Agents critique each other’s conclusions and iterate before delivering final intelligence.

## 🛠️ Tech Stack

- **Frontend**: Next.js 15 + Tailwind CSS + shadcn/ui + Recharts
- **Multi-Agent Framework**: CrewAI / LangGraph
- **LLMs**: Grok-3 + Claude 3.5 Sonnet
- **Blockchain Data**: Helius (Solana), DexScreener, Birdeye, Etherscan APIs
- **Social Intelligence**: X API + real-time monitoring
- **Vector Database**: ChromaDB / Pinecone (for narrative memory)
- **Backend**: FastAPI
- **Database & Auth**: Supabase
- **Deployment**: Vercel
